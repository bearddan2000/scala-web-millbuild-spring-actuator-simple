# scala-web-millbuild-spring-actuator-simple

## Description
A POC for actuator in springframework.

## Tech stack
- scala
- millbuild

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`
Available at:
- http://localhost
- http://localhost/health
- http://localhost/info

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
